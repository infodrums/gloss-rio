Glóssario de Programação
Servidor web
Servidor web é um computador responsável pelo armazenamento, processamento e entrega dos arquivos dos sites para os navegadores.
Servidor DNS
Um Servidor DNS é um sistema responsável por converter a url de um site para o endereço IP.
Dominío
Um Dominío é um endereço para o site, sendo uma parte da url.
Hospedagem
Hospedagem é um serviço que permite a publicação de um site na internet.
